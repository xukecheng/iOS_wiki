Instance Subscript

# subscript(_:)

Returns the context’s value for a key, or a the key’s default value if the
context doesn’t define a value for the key.

7.0+  visionOS 1.0+

    
    
    subscript<Key>(key: Key.Type) -> Key.Value where Key : PreviewContextKey { get }
